A modern, framework-aware duty logging system for FiveM. Track on/off-duty events per job with multiple logging backends and real-time state management.

## ✨ Features

* **Multi-Framework Support:** Works out of the box with ESX, ESX Legacy, QBox, and Standalone servers.
* **Multiple Loggers:** Log duty events via ESX Discord, ox_lib, or direct Discord webhooks.
* **Real-Time StateBags:** Access duty state from any script using modern state management.
* **Developer Friendly:** Versatile exports included to easily integrate duty tracking into your scripts.
* **Multi-Language:** Built-in locale support for English and Dutch.
